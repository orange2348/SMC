The module contains an example Java application which implements the TCP state machine on top of a UDP socket. This implementation represents TCP only at its most fundamental, not attempting to capture the more sophisticated TCP aspects.

The TCP state machine is defined in {@code TcpConnection.sm} and the SMC plug-in is used to generate the source file {@code TcpConnectionContext.java}. Once built, the application can be run in either server mode or client mode. In server mode, the command line is: 

$ java -classpath ... smc_ex6.server <port>
where <port> is the server TCP port.

The client command line is similar: 

$ java -classpath ... smc_ex6.client <host> <port>
where <host> is either a host name or dotted IP address and <port;> is the server TCP port opened on that host.
